# Calculator App 🧮

A sleek and modern calculator web application built with HTML, CSS, and JavaScript. This calculator performs basic arithmetic operations with a beautiful dark-themed interface.

## ✨ Features

- **Basic Operations**: Addition (+), Subtraction (-), Multiplication (*), Division (/)
- **Percentage Calculation** (%)
- **Clear Functions**: AC (All Clear) and DEL (Delete)
- **Decimal Point** support (.)
- **Double Zero** (00) for quick input
- **Responsive Design** - works on different screen sizes
- **Modern UI** with gradient background and glass-morphism effects
